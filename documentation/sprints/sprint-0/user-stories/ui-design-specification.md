# 软件规模评估系统 - UI设计规范文档

## 一、设计原则和理念

### 1. 设计指导原则
#### 政府部门适用性原则
- **权威性强：** 界面设计体现政府机构的权威性和严肃性
- **规范统一：** 严格按照政府信息化标准和规范
- **安全可信：** 界面设计传达系统的安全性和可信度

#### 用户体验原则
- **简洁明了：** 界面布局清晰，信息层次分明
- **操作高效：** 减少用户操作步骤，提高工作效率
- **容错友好：** 提供清晰的错误提示和操作指导

#### 专业性原则
- **数据精准：** 所有数据展示精确到位，支持审计追溯
- **功能完备：** 满足政府评估工作的所有专业需求
- **标准合规：** 100%符合《长沙市财政评审中心政府投资信息化项目评审指南》

### 2. 视觉设计理念
```
政府蓝 + 专业灰 + 清新白 = 权威 + 专业 + 清晰
```

## 三、字体规范

### 1. 字体家族
```
中文字体栈：
Primary: "PingFang SC", "Microsoft YaHei", "Hiragino Sans GB", sans-serif
Fallback: "SimHei", "Arial Unicode MS", sans-serif

英文数字字体栈：
Primary: "SF Pro Display", "Segoe UI", "Roboto", sans-serif
Fallback: "Arial", "Helvetica Neue", sans-serif

等宽字体 (代码/数据)：
Primary: "SF Mono", "Consolas", "Monaco", monospace
Fallback: "Courier New", monospace
```

### 2. 字体大小规范
```
标题层级 (Heading Scales)：
H1 主标题：24px (1.5rem) - 页面主标题
H2 二级标题：20px (1.25rem) - 区域标题
H3 三级标题：18px (1.125rem) - 模块标题
H4 四级标题：16px (1rem) - 子模块标题
H5 五级标题：14px (0.875rem) - 表格标题

正文层级 (Body Scales)：
大号正文：16px (1rem) - 重要说明文字
标准正文：14px (0.875rem) - 常规正文内容
小号正文：12px (0.75rem) - 辅助说明文字
标签文字：10px (0.625rem) - 标签、状态标识

数据展示：
大数字：32px (2rem) - 重要数据展示 (如总费用)
中数字：24px (1.5rem) - 统计数据
小数字：18px (1.125rem) - 表格数据
```

### 3. 行高和间距
```
行高规范 (Line Height)：
标题行高：1.2 - 标题类文字
正文行高：1.5 - 正文内容
密集行高：1.3 - 表格、列表
松散行高：1.8 - 长篇说明文字

字间距规范：
标题字间距：-0.025em
正文字间距：0em
数字字间距：0.05em
```

## 七、交互规范

### 1. 状态反馈
```
加载状态 (Loading States)：
按钮加载：显示转圈图标，按钮禁用
页面加载：显示骨架屏或进度条
数据加载：显示加载动画

悬停状态 (Hover States)：
按钮悬停：背景色变化，0.2s过渡
链接悬停：颜色变化，添加下划线
卡片悬停：阴影加深

点击状态 (Active States)：
按钮点击：按钮稍微下沉效果
列表项点击：背景色短暂高亮
```

### 2. 动画规范
```
过渡动画 (Transitions)：
颜色变化：0.2s ease
尺寸变化：0.3s ease-in-out
位置变化：0.3s cubic-bezier(0.4, 0, 0.2, 1)

页面切换：
淡入淡出：0.3s ease
滑动切换：0.4s ease-in-out

加载动画：
旋转加载：1s linear infinite
进度条：根据实际进度显示
```

### 3. 响应式断点
```
断点定义 (Breakpoints)：
sm: 640px   - 小型设备 (手机横屏)
md: 768px   - 平板设备 (iPad)
lg: 1024px  - 笔记本电脑
xl: 1280px  - 桌面显示器
2xl: 1536px - 大型显示器

设计适配：
<768px：单列布局，堆叠式
768px-1024px：两列布局，折叠导航
>1024px：完整布局，固定导航
```

## 八、特殊设计要求

### 1. 政府系统特殊要求
```
权威性设计：
- 使用政府标准蓝色作为主色调
- 页面头部展示政府机构标识
- 重要按钮使用红色印章样式图标
- 审批流程使用公文式的严肃设计

安全性设计：
- 敏感数据使用特殊背景标识
- 操作按钮需要二次确认
- 重要页面添加水印或防截图提示
- 登录界面使用安全验证码

专业性设计：
- 数据表格使用专业的统计样式
- 计算过程使用公式展示格式
- 报告界面使用公文格式排版
- 审批意见使用正式公文样式
```

### 2. 无障碍设计要求
```
颜色对比度：
- 正文文字对比度 ≥ 4.5:1
- 大号文字对比度 ≥ 3:1
- 图标对比度 ≥ 3:1

键盘导航：
- 所有交互元素支持Tab键导航
- 聚焦状态有明确的视觉反馈
- 提供快捷键支持

屏幕阅读器：
- 所有图片提供alt属性
- 表格提供完整的标题结构
- 表单提供明确的标签关联
```
