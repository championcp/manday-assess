## 🏗️ 系统架构分析

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **UI库**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia Store
- **构建工具**: Vite

### 页面路由结构
```
/                    -> HelloView (首页)
/home               -> HomeView 
/about              -> AboutView
/projects           -> ProjectListView (项目列表)
/projects/create    -> ProjectFormView (新建项目)
/projects/:id       -> ProjectDetailView (项目详情)
/projects/:id/edit  -> ProjectFormView (编辑项目)
/projects/:id/calculate -> NesmaCalculateView (NESMA计算)
```
