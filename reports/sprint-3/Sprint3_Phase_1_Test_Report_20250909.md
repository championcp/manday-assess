## ⚠️ 发现的问题和建议

### 1. 技术问题 (优先级: 中)
**问题**: HelloView中使用`window.location.href`导航
```javascript
// 当前实现
const navigateToDemo = () => {
  window.location.href = '/projects'
}

// 建议改进
const navigateToDemo = () => {
  router.push('/projects')
}
```
**影响**: 导致页面完全刷新，失去SPA优势

### 2. 工具连接问题 (优先级: 低)
**问题**: Chrome MCP服务连接不稳定
**状态**: 已采用替代测试方案
**影响**: 需要手动进行UI交互测试

### 3. 后端依赖问题 (优先级: 高)
**问题**: Spring Boot启动时间过长
**原因**: 首次启动需编译测试类
**影响**: API集成测试延迟
